Othello Game




How To Play:

-Can only place pieces on a tile where there is adjacent piece belonging to the opponent

-Must enclose opponent horizontally, vertically and/or diagonally

-Person with most pieces at end of the game is the winner

-If using safari, for the save and load feature, make sure Local File Restrictions are disabled under the Develop menu and refresh the page.
